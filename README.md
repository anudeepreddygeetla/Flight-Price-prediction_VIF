# Flight Price Prediction

This project predicts flight ticket prices using machine learning techniques.

## Features Used
- Airline
- Source
- Destination
- Duration
- Total Stops
- Days Left to Departure
- Seat Class

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Varience Inflation Factor

## Goal
To analyze flight data and predict ticket prices accurately.
